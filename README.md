# UnraidIcons

A polished collection of custom icons for Unraid Docker containers.

This repository provides high-quality PNG icons designed to improve the visual appearance of applications in the Unraid Docker interface. It is intended as a simple place to store and share custom icons that can be used across self-hosted services and containerized apps.

## Features

- Custom icons tailored for Unraid Docker containers
- PNG format for broad compatibility
- Useful for personalizing and improving your Unraid dashboard
- Lightweight and easy to use

## Usage

To use an icon from this repository in Unraid:

1. Open the repository and select the icon you want to use.
2. Copy the raw image URL or download the PNG file directly.
3. In Unraid, edit the Docker container you want to customize.
4. Paste the icon URL into the **Icon URL** field, or upload the image if preferred.
5. Save the container settings.

## About

This repository is meant to serve as a growing collection of clean, reusable Docker icons for the Unraid community. Whether you're customizing your own server or looking for icons for less common applications, this project aims to make that process easier.
